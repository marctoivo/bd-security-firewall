# BD Security Firewall

A comprehensive WordPress security plugin providing enterprise-grade protection.

## Features

- **Web Application Firewall (WAF)** — SQL injection, XSS, and request filtering
- **Brute Force Protection** — Login attempt limiting with configurable lockout
- **Two-Factor Authentication** — TOTP (authenticator app) and Email OTP
- **Geo-Blocking** — Block or allow traffic by country
- **File Integrity Monitoring** — Detect unauthorized file changes
- **Activity Logging** — Track logins, content changes, plugin updates
- **Security Headers** — HSTS, X-Frame-Options, Content-Type-Options
- **Login Security** — Hide WP version, disable XML-RPC, block PHP uploads

## Requirements

- WordPress 5.8+
- PHP 7.4+

## Installation

1. Upload the plugin to `/wp-content/plugins/bestdid-security/`
2. Activate through the WordPress admin
3. Configure settings under **BD Security** in the admin menu
